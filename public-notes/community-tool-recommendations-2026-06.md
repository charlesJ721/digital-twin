## 未接入但值得关注

### 强烈推荐

**1. Gamma.app** — AI 演示文稿/文档生成
- 一句话 → 完整 pitch deck、产品 spec、微网站
- PM 频繁需要 stakeholder-ready 材料
- gamma.app，免费层可用

**2. Lovable / Bolt.new** — Vibe Coding 平台
- 描述 → 全栈 Web 应用，几分钟出原型
- 户型标注网页、内部工具等想法速出原型
- lovable.dev / bolt.new，免费层

### 值得关注

**3. Stash** (alash3al/stash, ⭐711)
- Agent 持久记忆层，Postgres 存储
- 解决跨 session 上下文丢失
- 单二进制部署，自托管

**4. PainSpotter** (archoor/painspotter-mcp)
- AI 分析 Reddit/HN/PH 用户痛点
- 自动化需求发现引擎
- MCP Server

**5. Heptabase** — 视觉化知识管理
- 白板+卡片+AI 连接，适合深度思考和战略分析
- 付费 $11.99/月

## 智能家居相关

- **Home Assistant**: 全屋智能核心平台，2,500+ 集成，完全本地
- **Homey Pro**: 平衡易用性和功能，视觉化 Flow builder
- **装修要点**: 预留中性线、加深底盒、预埋网线、选 Matter 协议设备

## 筛选标准

筛掉了：纯开发者工具、需复杂自托管的基础设施、和已有工具重叠的、纯 AI 开发框架。
保留了：PM 能用上的、一条命令可装的、能闭合「想法→行动」的。
